`bctl version` is used to get the version of the blueprint CLI.

## Usage

```bash
bctl version [flags]
```

## Flags

| Flag | Description | Default |
| ---- | ----------- | ------- |
| `-h, --help` | Display the help for update |
| `--short` | Display version in short format | `false` |
